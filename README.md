# VM_Visualizer
This project will basically be C++ simulator of Virtual Memory in Operating Systems. I will also use raylib for the first time to create an interactive step-by-step walkthrough of Virtual Memory. 

# Motivation
 EECS 370: Computer Organization was definitely my most enjoyable class at Umich. We build projects on pretty much everything we learned in class including Assemblers, Linkers, CPU Pipelined Processors, and Caches. However, we never built a project on Virtual Memory. We learned Virtual Memory, took a heavy exam on it too, but we never really got to implement it ourselves. What a drag, right! Given that Virtual Memory feels like a hard concept at first and has a lot of factors that play into it, I'm going to fulfill my own interest as well as help my fellow EECS 370 students with this VM Visualizer. Hope you enjoy and always always LOVE EECS 370!!! 

# Goals
- Build a virtual memory simulator in C++
- Use raylib to create interactive graphical interface
- Show address translation step-by-step
- Visualize components such as:
    - virtual addresses
    - page tables
    - TLBs
    - physical memory
    - page faults
    - replacement policy
